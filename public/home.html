Hello, Developers

Where Developers Connect

Entering a new industry is not only challenging but also scary. Having the proper resources and connections can make all the difference. With this in mind, we being new developers, came up with an idea in hope to help minimize and overcome those challenges. Welcome to Dev Hangout, which allows developers across the country to connect, share, & collaborate on their project ideas.

Future Coming Soon!

  • Map feature allowing users to locate local developers to connect with.
  • Forum where users can share ideas, solutions, & resources.

Application Functionality

Currently, our application allows users to create a profile, chat with anyone who is registered to this application, & view their Github (only if you provided us with your Github username during the sign-up process) profile info.
